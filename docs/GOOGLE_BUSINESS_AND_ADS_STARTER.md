In the first week:

- Ask a small number of real families or professional partners for honest reviews if allowed and appropriate.

## 5. Google Ads approach for AZOE

For a first campaign, start with a simple Search campaign, not a broad multi-format campaign.

Why:

- Families already searching for care are high-intent.
- Text ads are simpler to manage than display or video.
- It is easier to control budget and search terms.

Primary goal:

- `Leads`

Best conversion actions to track:

- Phone calls from ads
- Clicks to call from the website
- Contact-page visits if call tracking is not ready yet

## 6. Link the Business Profile to Google Ads

After the Google Business Profile exists, link it in Google Ads so address and local business details can support the ads.

This also helps with location assets and local trust signals.

## 7. First campaign structure

Campaign type:

- `Search`

Geographic targeting:

- Start tight:
  - Lacey
  - Olympia
  - Tumwater
  - nearby Thurston County searches

Budget:

- Start small and measurable.
- Suggested learning budget: `\$20-\$35/day`

Ad schedule:

- Only run during hours when someone can answer the phone or return inquiries quickly.

Bid strategy:

- Start with a lead-focused setup Google recommends during campaign creation.
- If manual choices appear overwhelming, use the simplest lead-focused option first and improve after data comes in.

## 10. What to watch in week 1 of ads

Check these every few days:

- Search terms people actually used
- Calls generated
- Cost per click
- Which headlines get impressions
- Whether irrelevant searches are slipping in

Make these early adjustments:

- Pause weak keywords
- Add negative keywords
- Narrow the service area if spend is drifting
- Rewrite any ad copy that sounds generic or misleading
