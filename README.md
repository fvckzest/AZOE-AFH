# AZOE Adult Family Home

Marketing site for AZOE Adult Family Home built with Vite, React, and Cloudflare Pages.

## Stack

- Vite
- React
- Cloudflare Pages

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Handoff preflight

Run this before a live launch or business handoff:

```bash
npm run handoff:check
```

## Cloudflare Pages

This project is a static marketing site intended to run on Cloudflare Pages.

Helpful Cloudflare commands:

```bash
npm run cf:project:create
npm run cf:deploy
```

The site now uses direct phone and email contact instead of an embedded inquiry form.
